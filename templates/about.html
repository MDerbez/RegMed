About RegMed

RegMed

Homecare Electronic Record System

Purpose

RegMed is a simple electronic health record (EHR) system designed specifically for homecare services. It helps healthcare providers manage patient information and medical records efficiently.

Features
  • Patient Management
  • Medical Records Tracking
  • Visit Documentation
  • Simple Dashboard
  • Mobile-Friendly Interface
CS50 Project

This application was developed as part of the CS50 course project. It demonstrates web development skills using Flask, HTML, CSS, and JavaScript.

Security Note
Demo Application: This is a demonstration version. Data is stored in memory and will be lost when the application restarts. For production use, implement proper database storage and security measures.
Technology Stack
  • Python (Flask)
  • HTML5
  • CSS3 (Bootstrap)
  • JavaScript
Getting Started

Begin by adding patients to the system, then document their medical visits and treatments.
